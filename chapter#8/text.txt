THIS PROGRAM UTILIZES DICTIONARIES AND FILE HANDLING, SO IF YOU ARE NOT FAMILIAR WITH THESE CONCEPTS YET, IT IS RECOMMENDED TO LEARN THEM FIRST. 
IN ESSENCE, THIS PROGRAM ENCRYPTS THE CONTENTS OF A FILE, AND A CORRESPONDING PROGRAM DECRYPTS THE ENCRYPTED FILE TO DISPLAY ITS CONTENTS ON THE SCREEN. 
THIS TASK IS BASED ON CHAPTER EIGHT, PROGRAMMING EXERCISE THREE, FROM TONY GADDIS BOOK.
